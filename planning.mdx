<Steps>
  <Step title="Kick-off do Projeto">
    <Card >
Data: [Definido pela lideranca do produto]<br/>
Participantes:  Todos os membros da equipe<br/><br/>
**Objetivo:**
<br/>
- Apresentar o escopo do projeto
- Definir a data final de entrega
- Dividir as principais responsabilidades entre os membros da equipe
- Estabelecer metas e expectativas claras
<br/>
**Tarefas:**
<br/>
- Revisar o escopo e requisitos do sistema
- Configurar as ferramentas de comunicação e gestão de tarefas Jira
</Card>
  </Step>
  <Step title="Definição das Entregas e Divisão de Tarefas">
      <Card >
Data         :  [Definido pela lideranca do produto]<br/>
Participantes:  Líder do projeto e membros-chave da equipe<br/><br/>

**Objetivo:**
<br/>
- Quebrar o projeto em entregas menores (sprints ou fases)
- Estabelecer prazos realistas para cada entrega
- Designar responsáveis para cada tarefa
<br/>
**Tarefas:**
<br/>
- Criar um cronograma detalhado no formato de uma linha do tempo
- Identificar dependências e possíveis gargalos
- Revisar recursos necessários para cada fase do projeto
</Card>
  </Step>
  <Step title="Reuniões de Progresso Semanais">
 <Card>
Data         :  Semanalmente<br/>
Participantes:  Toda a equipe<br/><br/>

**Objetivo:**<br/>
<br/>
- Monitorar o andamento das tarefas e entregas
- Identificar bloqueios e resolvê-los rapidamente
- Ajustar o planejamento conforme necessário
<br/>
**Tarefas:**
<br/>
- Cada membro da equipe reporta o que fez no dia/semana anterior e o que pretende fazer no próximo período
- Discutir problemas e encontrar soluções em conjunto
- Atualizar o status das entregas no sistema de gestão de tarefas
</Card>
  </Step>
    <Step title = "Revisões de Entregas (Final de Cada Fase)">
<Card>
Data         : Final de cada fase do projeto<br/>
Participantes:  Líder do projeto e equipe de desenvolvimento/testes<br/><br/>

**Objetivo:**
<br/>

- Avaliar o que foi concluído
- Realizar testes de validação das entregas feitas
- Identificar melhorias e pontos de ajuste antes da próxima fase
<br/>
**Tarefas:**
<br/>
- Revisar o código, funcionalidades e testes automatizados
- Avaliar o feedback dos stakeholders (se aplicável)
- Priorizar ajustes e correções para a próxima fase
</Card>
  </Step>
  <Step title = "Fase Final e Entrega">
  <Card>
Data         :  Última semana antes da data final de entrega<br/>
Participantes:  Toda a equipe<br/><br/>

**Objetivo:**
<br/>
- Realizar uma verificação final do sistema
- Garantir que todas as entregas estejam completas e funcionais
- Preparar o ambiente de produção para a entrega final
<br/>
**Tarefas:**
<br/>
- Realizar testes finais de aceitação
- Documentar o sistema e treinamentos (se necessário)
- Preparar a entrega para o cliente ou stakeholders
</Card>
   </Step>
   <Step title = "Reunião de Retrospectiva">
   <Card>
Data         :  Após a entrega final<br/>
Participantes:  Toda a equipe
<br/>
**Objetivo:**
<br/>
- Avaliar o processo de desenvolvimento e entrega
- Identificar pontos fortes e áreas de melhoria para futuros projetos
- Registrar aprendizados
<br/>
**Tarefas:**
<br/>
- Discussão aberta sobre o que funcionou bem e o que pode ser melhorado
- Documentar sugestões e lições aprendidas
- Celebrar as conquistas da equipe
</Card>
  </Step>
</Steps>
